# Projects

A collection of personal projects built to apply what I'm learning, explore new ideas, and create things I'm proud of.

## How It's Made

**Tech used:** JavaScript, HTML, CSS *(varies by project)*

Each project in this repo starts with a goal — something to build, a problem to solve, or a concept to explore hands-on. Rather than following a tutorial step-by-step, these are things I've designed and built myself, making decisions along the way about structure, styling, and functionality.

Every project is an opportunity to connect the dots between what I've learned and how it actually works in practice.

## Optimizations

Across projects, I focus on:

- Writing semantic, accessible HTML from the start
- Keeping CSS organized and avoiding redundancy
- Structuring JavaScript in a way that's readable and easy to extend
- Revisiting older projects to refactor as my skills improve

## Lessons Learned

- Building real things reveals gaps in understanding that exercises alone don't
- Scope creep is real. Starting small and shipping something complete beats planning a perfect project that never gets done
- Design and functionality are both part of the work; a project that works but looks rough is still unfinished
- Every project teaches something new, even when it doesn't go as planned
